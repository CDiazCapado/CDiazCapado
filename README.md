## About

I build production systems. My foundation is **software engineering with deep expertise in test automation and DevOps** — architecting CI/CD pipelines, designing automation frameworks, and shipping reliable software at scale. That same engineering rigor is what I now bring to **MLOps and AI Engineering**.

The stack evolved; the mindset did not. Testable, observable, deployable, reproducible — the principles that make infrastructure reliable are the ones that make ML systems production-ready. Most ML projects fail not because the model is weak, but because the engineering around it is. That gap is where I work.

I'm currently building **NomadOptima**, a Learning-to-Rank recommender system with explainable outputs, alongside a portfolio of AI projects exploring multi-agent systems, computer vision pipelines and fine-tuned LLMs. All of them with the same non-negotiable: engineered like production, not like a notebook.
